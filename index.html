How to Join

  1. 1 Open Minecraft and select Multiplayer (Java) or Servers (Bedrock).
  2. 2 Click "Add Server" and enter the details provided above.
  3. 3 Save and Click "Join Server".
  4. Note: Ensure the server is currently started via the "Start Server" instructions.

Create Account

2

3

Request Access

Message a staff member or in the support channel:
"My Aternos username: YOUR_USERNAME"

4

Wait for Approval

Staff will grant you permission to start the server manually.

Start & Play
